# 二胡识谱

一款帮助二胡学习者练习识谱和音准的 iOS 教学应用。

## 功能

- **简谱显示**：支持数字简谱展示，包含高低八度标记
- **实时音高检测**：通过麦克风捕捉演奏，实时显示频率和音符
- **音准判断**：将演奏音高与乐谱对比，给出视觉反馈
- **内置曲库**：包含多首经典二胡曲目
- **演奏统计**：展示正确率和演奏进度

## 技术实现

- Swift 6 + SwiftUI
- AVAudioEngine 实时音频处理
- 自相关算法音高检测
- 音分偏差精度计算

## 开始使用

1. 用 Xcode 16+ 打开 `Package.swift`
2. 选择 iOS 18+ 模拟器或真机
3. 构建并运行
